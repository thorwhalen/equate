equate.registry

A tiny name-keyed strategy registry with lazy factories (open-closed dispatch).

Each pipeline stage (featurize, compare, block, match, cluster) owns a Registry mapping a string name to a lazy factory that builds the strategy on demand, so a heavy optional import fires only when that strategy is actually used. Users select a strategy by name (featurize='tfidf') or pass a callable directly — Registry.resolve() accepts both, and users can register their own strategies without subclassing.

class equate.registry.Registry(kind: str)[source]

A name -> lazy factory registry for one pipeline stage’s strategies.

>>> reg = Registry('demo')
>>> @reg.register('double')
... def _double_factory():
...     return lambda x: x * 2
>>> reg.names()
['double']
>>> reg.create('double')(21)          # look up + build + call
42
>>> reg.resolve(str.upper)('hi')       # a callable passes through unchanged
'HI'
>>> 'double' in reg
True

meta(name: str) → Any[source]

Return the declared metadata for name (None if none was declared).

Raises KeyError for an unknown name, consistent with factory() — so an unknown name is never silently conflated with a known name that has no meta.

names() → list[source]

Sorted list of registered strategy names.

register(name: str, factory: Callable | None = None, *, meta: Any = None)[source]

Register factory under name. Usable as a decorator or directly.

The factory is called lazily (by create()/resolve()), so any heavy import inside it fires only on use. meta is optional declared metadata (e.g. a FeaturizerMeta/ComparatorMeta).

resolve(spec, *args, **kwargs)[source]

Resolve spec to a strategy.

A callable spec passes through unchanged; a str is looked up and its factory is called with (*args, **kwargs).
